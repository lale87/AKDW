%---------------------------------- 
\subsection{Persistent}
%---------------------------------- 
Persistentes XSS zielt darauf ab, Schadcode in Form einer Skript-Routine
dauerhaft auf einem Webserver zu speichern.
Um den Schadcode auf den Server zu übertragen werden Schnittstellen benutzt, die
es einem Nutzer erlauben, eigene Eingaben an einen Server zu senden, die in der
späteren Darstellung einer Webseite genutzt werden. Der hier eingebene Text,
wie er z.B. in Diskussionforen häufig vorkommt, kann neben dem eigentlichen Text
um Skript-Elemente erweitert werden \cite{fSidI13}. Werden die übertragenen
Eingabedaten nicht überprüft, so wird der Schadcode dauerhaft in die Seite
eingebettet. Jedem Besucher der so infizierten Webseite wird vom vermeintlich
sicheren Server nun der Schadcode ausgeliefert, der vom Browser des Nutzers
interpretiert und ausgeführt wird, sofern dies nicht speziell vom Client
unterbunden wird \cite[1772]{GANA07}.
