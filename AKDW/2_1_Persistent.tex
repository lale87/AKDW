%---------------------------------- 
\subsection{Persistent}
%---------------------------------- 
Persistentes XSS zielt darauf ab, Schadcode in Form einer Skript-Routine
dauerhaft auf einem Webserver zu speichern.
Um den Schadcode auf den Server zu übertragen, werden Schnittstellen benutzt,
die es einem Nutzer erlauben, eigene Eingaben, die in der
späteren Darstellung einer Webseite genutzt werden, an einen Server zu senden.
Der hier eingegebene Text, wie er z.B. in Diskussionsforen häufig vorkommt, kann
neben dem eigentlichen Text um Skript-Elemente erweitert werden \cite{fSidI13}. Werden die übertragenen
Eingabedaten nicht überprüft, wird der Schadcode dauerhaft in die Seite
eingebettet. Jedem Besucher der so infizierten Webseite wird vom vermeintlich
sicheren Server nun der Schadcode ausgeliefert, der vom Browser des Nutzers
interpretiert und ausgeführt wird, sofern dies nicht speziell vom Client
unterbunden wird \cite[1772]{GANA07}.
